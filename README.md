Bu projeyi Ubuntu 20.04.6, ROS Noetic, TurtleBot3 Noetic versiyon Waffle modelini kullanarak gerçekleştirdim. 2023-2024 Eğitim döneminde yaptığım eski çalışmalarımdan kalan bazı bilgiler ve hızlı bir tekrarla dokümanı inceleyerek projeye başladım. Launch ve script dosyalarında kod parçalarını yorum satırlarıyla olağanca ve öğrenebildiğimce açıklamaya çalıştım. Eksiklerim ve hatalarım için kusura bakmayınız. Hem zip dosyasında hem de github reposunda projeyi sizlerle paylaşıyor olacağım. Bahsi geçen eski çalışmalarımı içeren github reposunu görmek isterseniz buradan ulaşabilirsiniz : https://github.com/mertbayir/-SubuTetraEgitimProjeleri . Değerlendirmeniz için teşekkürler.
